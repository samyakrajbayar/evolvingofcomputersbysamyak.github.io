SECOND GENERATION COMPUTERS
(1959-1964)


In the late 1950s, the Second Generation Computers were introduced.
Second Generation computers used transistors in place of vacuum tubes. The transistors were far superior than vacuum tubes that allowed computers to become smaller, faster, cheaper, and more reliable than the first generation computers. The second generation computers used Assembly language instead of Machine language IBM 140, MARK III, and LEO are the examples of this generation of computers.


Features of Second Generation Computers:


  • The size was smaller than the first generation computer
  • The operating speed was faster
  • The cost got reduced.

Drawbacks of Second Generation Computers:


  • They produced a lot of heat
  • The speed was still slow
  • The memory was less.
